Library Management System

A Library Management System is a software built to handle the primary housekeeping functions of a library. Libraries rely on library management systems to manage asset collections as well as relationships with their members. Library management systems help libraries keep track of the books and their checkouts and profiles.

Library management systems also involve maintaining the database for entering new books and recording books that have been borrowed with their respective due dates.

There are two types of users.

Member user

Any library member can search database for books by their title or author, get more details on a book, borrow a book, renew a book, return a book, have a profile with the books he borrowed.

Does not have permission to see any librarian templates.

Librarian users

Can do everything member users do and also have access to logs, administration, add book.

On logs template can see logs up to most recent database changes.

On administration page librarians can view details of any book, edit details of any book, delete any book or add a new book.

Add book functionality uses Google Books API to retrieve data from a given book title.
